Cara Upload Foto

Panduan Mudah Upload Foto ke Galeri

Dengan mengikuti langkah-langkah di bawah ini, foto kamu akan langsung muncul di Discord channel kami dan bisa dilihat oleh seluruh komunitas!

  1. Klik tombol "Upload Foto" di bagian atas halaman.
  2. Pilih file foto dari perangkatmu. Pastikan ukuran file tidak lebih dari 10 MB.
  3. Foto akan langsung dikirim ke server Discord komunitas.
  4. Tim kami akan meninjau dan menambahkan fotomu ke galeri utama di website ini.
  5. Selesai! Kamu sudah berkontribusi untuk galeri komunitas kita. Terima kasih!
